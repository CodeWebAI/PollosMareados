# 🐔 Pollos Mareados - Sitio Web

Página web oficial de **Pollos Mareados** - Los Auténticos Pollos a la Leña

## 🚀 Tecnologías

- **React 19** - Framework de JavaScript
- **Vite 8** - Build tool ultra-rápido
- **Tailwind CSS v4** - Framework de CSS utility-first
- **pnpm** - Gestor de paquetes eficiente

## 🎨 Colores del Proyecto

El sitio utiliza los colores del logo oficial:

- **Rojo Principal**: `#E63946` (`pollos-red`)
- **Rojo Oscuro**: `#C62828` (`pollos-red-dark`)
- **Amarillo**: `#FFD60A` (`pollos-yellow`)
- **Amarillo Oscuro**: `#FFAA00` (`pollos-yellow-dark`)
- **Naranja**: `#FF6B35` (`pollos-orange`)

## 📦 Instalación

```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm dev

# Construir para producción
pnpm build

# Vista previa de producción
pnpm preview
```

## 🏗️ Estructura del Proyecto

```
src/
├── components/
│   ├── Header.jsx          # Navegación principal
│   ├── Hero.jsx            # Sección hero/portada
│   ├── PlaceholderSection.jsx  # Plantilla para secciones futuras
│   └── Footer.jsx          # Pie de página
├── App.jsx                 # Componente principal
├── main.jsx               # Punto de entrada
└── index.css              # Estilos globales y configuración Tailwind
```
